# VickyBytes - Live Event Streaming Platform

VickyBytes is a premium, responsive Live Event Streaming Platform UI built with modern web technologies. This project focuses on high-quality UI/UX, smooth animations, and scalable frontend architecture.

## 🚀 Features

- **Event Listing Page**: A dynamic grid of live and upcoming events with category-based filtering.
- **Event Streaming Page**: A dual-column layout featuring a responsive video player, rich event description, and a real-time simulated live chat interface.
- **Premium Design Aesthetics**: Deep dark-mode theme with vibrant purple accents, modern typography (Geist Sans/Mono), glassmorphism effects, and micro-interactive hover states.
- **Fully Responsive**: Optimized layouts that seamlessly adapt from mobile phones to 4K desktop screens.
- **Smooth Animations**: Integrated `framer-motion` for layout transitions and staggered grid animations.

## 🛠️ Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS (v4 structure)
- **Animations**: Framer Motion
- **Icons**: React Icons (Lucide/Heroicons standard)
- **Utilities**: clsx, tailwind-merge

## 📦 Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd VickyBytes
   ```

2. **Install dependencies**:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open in browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)
